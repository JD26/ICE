This folder contains an uncompressed version of http://en.cppreference.com site's offline archive (dated from 02-08-2014, downloaded from http://en.cppreference.com/w/Cppreference:Archives).
Some files where addeed: make_index (a bash script to extract titles from all html pages), en/zinjai_index (list of html files and their respective titles for searching), and all svg graphics where converted to png.
